# toutiao-m

### 介绍

toutiao-m 是基于 vue 的移动端项目，所使用到的技术有

1. vue vue-Router vueX
2. axios
3. vant-ui

### 基本功能

1. 登录
2. 浏览、搜索文章
3. 浏览、管理自己的信息
4. 关注用户，收藏、点赞喜爱的文章



该项目可以帮助你了解

1. 代码封装
2. 样式穿透

### 下载依赖

```
npm install
```

### 启动
```
npm run serve
```

### 浏览

在浏览器中键入 http://localhost:8080 即可预览项目
